# クイズで暗記【百人一首】 

百人一首を楽しく暗記できる無料のWebアプリケーションです。

## 🌸 特徴

- **音声読み上げ機能**: 上の句と下の句を自動で読み上げ
- **カテゴリ別学習**: 20首ごとに分けて効率的に学習
- **進捗管理**: 最高記録、コンボ数を保存
- **カスタマイズ可能**: 効果音、アニメーション、自動読み上げのON/OFF
- **レスポンシブデザイン**: スマホ・タブレット・PCに対応
- **伝統的デザイン**: 桜の花びらアニメーション、和風UI

## 🚀 使い方

1. 学習したい範囲をカテゴリから選択
2. 「修行を始める」ボタンをクリック
3. 上の句を読んで、正しい下の句を選択
4. 結果を確認して、さらに学習を深める

## 📱 動作環境

- モダンブラウザ（Chrome, Firefox, Safari, Edge）
- JavaScript有効
- Web Speech API対応（音声読み上げ機能）

## 🛠️ 技術スタック

- HTML5
- CSS3（カスタムアニメーション）
- Vanilla JavaScript
- Web Speech API
- Local Storage
